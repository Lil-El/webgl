## 矢量、矩阵

| 类别 | 类型                     | 描述                                        |
| ---- | ------------------------ | ------------------------------------------- |
| 矢量 | vec2~4、ivec2~4、bvec2~4 | 具有 2、3、4 个浮点数、整型、布尔元素的矢量 |
| 矩阵 | mat2~4                   | 2x2、3x3、4x4 的浮点数元素的矩阵            |

```
矢量类型：vec4 = vec4(1.0, 2.0, 3.0, 4.0)
矩阵类型：
      [3  7  1]
      [1  5  3]
      [4  0  7]
```
